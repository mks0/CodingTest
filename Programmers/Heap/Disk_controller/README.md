## 분류
힙
## 언어
C++17
## 주의사항
해당사항 없음
## 접근 방법
* 벡터에 저장된 작업들에 대해 요청시간 기준으로 오름차순 정렬
* 우선순위 큐 선언 - 작업 소요시간 오름차순 기준
* 벡터 작업들을 모두 우선순위 큐에 넣었고, 우선순위 큐가 비어 있을 때까지 반복
  + 추가 가능 작업들을 큐에 추가
  + 큐의 Top 작업이 완료 가능한 경우
    - 완료 기준으로 시간 증가 및 요청-종료까지의 시간 저장
  + 큐의 Top 작업이 완료 가능하지 않은 경우
    - 단일 시간 증가
