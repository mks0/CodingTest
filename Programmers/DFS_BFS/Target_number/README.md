## 접근 방법
* 최초 접근
  + 배열을 순회하며 누적 스택 내 요소 별로 연산 처리 후 다시 누적 스택에 push
  + 성능은 제일 떨어짐  
* DFS 재귀접근
  + 코드 직관적, 성능도 좋음
* DFS 비재귀접근
* BFS로 풀어도 무방할 것으로 보임.
