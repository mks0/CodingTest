# 베스트앨범
## 문제 링크
[프로그래머스 베스트앨범]
(https://programmers.co.kr/learn/courses/30/lessons/42579?language=cpp)
## 분류
해시
## 언어
C++17
## 주의사항
* 해당 장르의 곡이 1곡만 존재하는 경우 고려
## 접근 방법
* 곡 구조체 정의
  + 멤버 변수 : 고유번호, 재생 수
* 장르 구조체 정의
  + 멤버 변수 : 재생 수, 최다 재생 상위 2곡
  + 멤버 함수 : 곡 추가(최다 재생 2곡 추출 기능 포함)
* 키는 장르명, 값은 장르 구조체인 해시 정의하여 곡 추가
* 우선순위 큐를 이용하여 장르 별 총 재생 수 삽입
* 우선순위 큐 순회
## 기타 사항
해당사항 없음
